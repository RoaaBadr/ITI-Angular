# Installation

- nodeJs (node -v)
- install typescript (npm i -g typescript) (tsc -v)
- install angular/cli (npm i -g @angular/cli) (ng v)

## Create New Project

- ng new `projectName`

## to run angular

-- ng serve
-- ng serve -o
-- npm start

## to create a component in angular cli

- ng g c componentName --skip-tests
